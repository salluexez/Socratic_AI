# Socratic AI Mobile

Flutter mobile app scaffold for the hackathon build of the Socratic AI Teaching Assistant.

## Included

- Onboarding screen based on the provided Stitch UI
- Auth entry screen
- Subject picker
- Socratic chat screen
- Progress dashboard
- Profile screen
- Mock data and reusable design tokens

## Run

```bash
cd apps/mobile
flutter pub get
flutter run
```
